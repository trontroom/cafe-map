# Cafe Map

渋谷エリアのカフェを地図上に表示・検索・登録できるWebアプリケーションです。

## 概要

- 地図上にカフェのピンを表示
- カフェの新規登録・編集・削除
- 営業時間・Wifi・電源・駐車場の情報表示
- コメント機能
- いいね機能
- Supabaseによるクラウドデータ共有

## 使い方

1. ブラウザで `index.html` を開く
2. 地図をクリックするとカフェ登録フォームが開く
3. カフェ名・住所・緯度・経度などを入力して「登録」
4. 地図上のピンをクリックすると詳細・コメント・いいねが表示される

## 技術

- HTML / CSS / JavaScript
- Leaflet.js（地図表示）
- Supabase（クラウドデータベース）
